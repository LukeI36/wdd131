Final Project Site Plan

Site Name

Site name: Transformers Database

Domain name: tf-database.com

The name represents a library of information that's focused on the transformers franchise

Site Purpose

The site aims to educate new fans about the history of the transformers franchise, while for more experienced fans, the website acts as a library, where all information about the transformers franchise can be found in one place.

Scenarios

Where do the transformers come from and what is their history?

Where can I find the latest news of the transformers franchise?
